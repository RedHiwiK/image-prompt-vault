# NOTICE

## Attribution

The prompts in `models/<model>/prompts.json` are aggregated from publicly visible posts and curated community repos. Each entry preserves:

- `author` - the original poster's handle (e.g. `@dotey`, `@BubbleBrain`)
- `source_url` - direct link to the original post
- `upstream_repo` - which intermediate awesome-* repo this entry was sourced through

## Takedown / re-attribution

If you are an author of any prompt in this vault and would like:
- Your prompt removed
- Your attribution corrected
- A different handle / link used

Open an issue with the `id` (12-char fingerprint) of the affected entries, or with the `source_url`. Removals are typically processed within 7 days.
